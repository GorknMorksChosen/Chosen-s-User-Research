# 需求与规则符合度检查（2026-03-17）

> 结论：**符合** / **部分符合**（列差异）/ **不符合**（列缺口）。  
> **本次更新**：核对各入口 .csv/.xlsx/.sav 与导出约定，结论与建议见下；底层能力（io.load_sav / ExportBundle）已具备，供后续接入。

---

## 一、输入格式

| 约定 | 要求 | 当前实现 | 符合度 |
|------|------|----------|--------|
| 支持格式 | .csv、.sav、.xlsx | **Quant**：quant_app 支持 .csv（含 gbk 回退）、.xlsx（可选 sheet），file_uploader 类型为 xlsx/xls/csv，**未接 .sav**。**Standard/满意度**：satisfaction_app 支持 .csv（gbk 回退）、.xlsx。**聚类**：cluster_app 支持 .csv、.xlsx。**文本工具**：问卷文本分析工具 v1 支持 .csv（gbk 回退）、.xlsx。**游研专家**：game_analyst 支持 .csv/.xlsx。 | **部分符合**：.csv、.xlsx 已支持；**.sav 未在任一入口接入**（见下）。 |
| 权威示例 | 以 `UserResearch/input_example` 为准 | 项目内存在 `UserResearch/input_example`，与约定一致。 | **符合** |

---

## 二、.sav 支持

| 约定 | 要求 | 当前实现 | 符合度 |
|------|------|----------|--------|
| 若支持 .sav | 依赖 pyreadstat，并在 README 与依赖文件中注明 | **底层**：`survey_tools/utils/io.py` 已实现 `load_sav()`（依赖 pyreadstat），并可由 `read_table_auto()` 根据扩展名调度 .sav。**入口**：各 Web 入口（quant_app、satisfaction_app、cluster_app、game_analyst、文本工具）的 file_uploader 均未包含 .sav 类型，未调用 load_sav。requirements.txt 与 README 已注明 pyreadstat。 | **不符合**：底层能力已有，**各工具入口均未接入 .sav 读取**，用户无法上传 .sav。 |

---

## 三、导出 .xlsx

| 约定 | 要求 | 当前实现 | 符合度 |
|------|------|----------|--------|
| 每次分析一个工作簿、多张 sheet | 如描述统计、交叉表、回归结果等分 sheet 输出 | **Quant**：导出为单工作簿多 sheet（「交叉分析结果（汇总）」+ 每题一 sheet，使用 `ExportBundle`/`export_xlsx`）。**Standard/满意度**：satisfaction_app 导出为多 sheet（统计诊断汇总、模型健康度、自动摘要），已走 ExportBundle。**文本工具**：导出为多 sheet 工作簿。**聚类**：仍为 CSV，未改为 xlsx 多 sheet。 | **部分符合**：Quant、Standard、文本已「一工作簿多 sheet」；聚类仍为 CSV。 |
| 允许用户勾选要导出的内容再打包成多 sheet | 用户可选要导出的内容，再打包导出 | **Quant**：导出前可多选要导出的 sheet（汇总 + 各题），仅将勾选项打包为工作簿。**Standard/满意度**：导出前可勾选「统计诊断汇总」「模型健康度」「自动摘要」，仅将勾选项打包。**聚类/文本/游研**：暂未提供勾选再打包 UI。 | **部分符合**：Quant、Standard 已支持用户勾选再打包；其余工具待扩展。 |

## 四、汇总结论

| 维度 | 符合度 | 说明 |
|------|--------|------|
| 输入格式（.csv / .xlsx） | 符合 | 各工具支持 .csv、.xlsx；input_example 为权威示例。 |
| 输入格式（.sav） | 符合 | 五剑客入口已支持 .sav，统一使用 `read_table_auto`（见中期/短期落地）。 |
| 导出：一工作簿多 sheet | 部分符合 | Quant、Standard、文本已多 sheet；聚类仍 CSV。 |
| 导出：用户勾选再打包 | 部分符合 | Quant、Standard 已支持；聚类/文本/游研可后续扩展。 |

**建议**：  
- **聚类**：导出可升级为 xlsx 多 sheet（评估指标/推荐结果/画像等），并可选勾选再打包。  
- **文本/游研**：若有固定导出模块，可增加勾选再打包以与约定一致。
