---

## 图例

- **已实现**：代码/入口中明确存在该能力，可直接验收。
- **部分实现**：有相关逻辑但范围不全（如只有一种子类型、或依赖特定题型）。
- **未实现**：当前项目未覆盖，暂不验收。

## 汇总：可验收范围（阶段 C 用）

以下为 **已实现** 或 **部分实现** 的对应关系，建议在阶段 C 做逐条映射与验收：

- **Quant**：差异关系（卡方、Fisher、ANOVA、Kruskal、Tukey、正态/方差齐性检查）；单选/多选交叉与频数；描述统计与题型识别。
- **Standard**：相关（IPA 用相关系数）；多元线性回归、VIF、信度（Cronbach α）。
- **游研专家**：数据体检（异常/离群/缺失）；因子分析；路径分析/SEM；信度；Kano/SHAP；K-Means 分群。
- **聚类**：缺失值策略；因子降维；多算法聚类（KMeans/GMM/Agglomerative）；手肘/轮廓；推荐策略与画像。
- **Text**：开放题语义分析、编码、关键词与情感、摘要与导出。

**未实现（暂不验收）**：数据标签/编码、生成变量、量纲与日期处理、单样本/配对 t、双因素方差与简单效应、分层卡方与拟合优度、配对/Friedman 非参数、偏相关、Logistic 回归、填空题/项目分析、效度、验证性因子、中介/调节与有调节的中介、功效分析等。
