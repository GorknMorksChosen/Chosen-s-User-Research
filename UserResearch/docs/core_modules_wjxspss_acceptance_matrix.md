## 3. 差异关系研究

| 清单条目 | 对应核心模块（及模块/入口） | 验收方式 | 验收状态 |
|----------|---------------------------|----------|----------|
| t 检验（独立样本多组→ANOVA/Kruskal） | Quant | 运行 Quant，选一道定量题 + 分组变量（多组），做交叉/分组差异；确认输出为 ANOVA 或 Kruskal-Wallis，且含事后比较（如 Tukey 或 Dunn）。 | 待验收 |
| 方差分析（单因素 + 事后比较） | Quant | 同上；确认单因素方差分析或 Kruskal-Wallis 的 p 值、效应量，及 Tukey HSD 或 Dunn 事后两两比较结果。 | 待验收 |
| 卡方检验（2×2、R×C、Fisher） | Quant | 运行 Quant，选两道单选题做交叉表；确认列联表、卡方检验 p 值；若为 2×2 且期望频数小，确认使用 Fisher 精确检验。 | 待验收 |
| 非参数（Kruskal-Wallis + Dunn） | Quant | 选定量题 + 分组（多组），当不满足正态/方差齐性时确认自动或可选 Kruskal-Wallis，且含 Dunn 事后比较。 | 待验收 |
