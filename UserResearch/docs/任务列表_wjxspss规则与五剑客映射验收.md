## 三、任务 B：目录 → 五剑客「已有对应」功能

### B.1 目录口径

以 **`UserResearch/功能可执行清单_参考wjxspss.md`** 的六大类为准（与 wjxspss 目录对应）：

1. 数据探索与格式  
2. 问卷数据清理  
3. 差异关系研究（t 检验 / 方差分析 / 卡方 / 非参数）  
4. 相关与影响关系研究（相关 / 线性回归 / Logistic）  
5. 问卷专属研究分析（单选多选 / 信度效度 / 因子 / 路径 / 中介调节等）  
6. 其他分析能力（聚类、文本分析等）

### B.2 五剑客与目录的「已有对应」关系（待执行时核对代码确认）

| wjxspss 目录大类 | 五剑客 | 已有对应功能（README 口径） |
|------------------|--------|-----------------------------|
| 1. 数据探索与格式 | Standard（部分） | 描述统计、IPA 所需基础统计 |
| | Quant（部分） | 题型识别、交叉表前的数据理解 |
| 2. 问卷数据清理 | 游研专家 / 聚类 | 异常值/离群值、缺失值策略（剔除/均值/中位数等） |
| 3. 差异关系研究 | **Quant** | 卡方、Fisher、ANOVA、Kruskal-Wallis、Tukey HSD 等 |
| 4. 相关与影响关系 | **Standard** | 相关（重要性）、多元线性回归、VIF、信度 |
| | 游研专家（部分） | 回归/路径/SEM、Kano/SHAP |
| 5. 问卷专属研究 | **Quant** | 单选/多选交叉与卡方、频数 |
| | **Standard** | 信度、回归驱动的诊断 |
| | **游研专家** | 因子分析、路径分析、结构方程、中介/调节（若有） |
| 6. 其他（聚类、文本） | **聚类** | 因子降维、K-Means/多算法分群、推荐策略、画像 |
