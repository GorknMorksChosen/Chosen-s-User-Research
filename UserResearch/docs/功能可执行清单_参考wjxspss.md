# 统计与 AI 功能可执行清单（参考 wjxspss）

> 本清单依据 `UserResearch/wjxspss.docx`（即《SPSSAU 问卷/量表数据分析方法与应用》）提取整理，正文已提取为 `UserResearch/wjxspss_extracted.txt` 供检索与迭代。项目优先覆盖以下能力。

## 1. 数据探索与格式

- [ ] 分析方法与数据格式说明（定类/定量）
- [ ] 探索数据特征（描述统计、分布）
- [ ] 定类数据探索分析
- [ ] 定量数据探索分析
- [ ] 数据分布之正态性分析（图示法、检验法、转换处理）
- [ ] 常用分析方法选择指引（定类/定类，定类/定量，定量/定量）

## 2. 问卷数据清理

- [ ] 数据标签设置
- [ ] 数据编码
- [ ] 异常值处理
- [ ] 生成变量、常用处理、量纲处理
- [ ] 科学计算、汇总处理、日期相关处理
- [ ] 标题处理及其他

## 3. 差异关系研究

- [ ] **t 检验**：正态与方差齐性检查；单样本 / 配对样本 / 独立样本 t 检验
- [ ] **方差分析**：单因素、双因素及多因素方差分析；简单效应分析
- [ ] **卡方检验**：2×2 四格表、R×C 列联表与多重比较、Fisher、配对卡方、分层卡方、卡方拟合优度
- [ ] **非参数检验**：单样本 Wilcoxon；Mann-Whitney；Kruskal-Wallis；配对 Wilcoxon；Friedman

## 4. 相关与影响关系研究

- [ ] **相关分析**：两变量相关、偏相关
- [ ] **线性回归**：模型与检验、适用条件、多重/逐步线性回归、哑变量线性回归
- [ ] **Logistic 回归**：二元、多分类、有序 Logistic

## 5. 问卷专属研究分析

- [ ] 单选题与多选题分析（频数统计、卡方检验等）
- [ ] 填空题分析
- [ ] 项目分析
- [ ] 效度分析（结构效度等）
- [ ] 信度分析（信度系数等）
- [ ] 验证性因子分析（步骤与实例）
- [ ] 路径分析
- [ ] 结构方程模型分析
- [ ] 中介效应分析（流程与实例、多重中介）
- [ ] 调节效应分析（步骤与实例）
- [ ] 有调节的中介分析

## 6. 其他分析能力（书中或平台提及）

- [ ] 聚类分析（样本聚类、指标聚类）
- [ ] 文本分析（开放题语义/编码/摘要）
- [ ] 功效分析、假设检验设计
- [ ] 荟萃分析、生存分析、时间序列分析、曲线回归等（按需扩展）
